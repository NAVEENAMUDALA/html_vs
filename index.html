What's the Difference ?

HTML

Hyper Text Mark up language 		Create the structure
  • Controls the layout of content
  • provides structure for web page design
  • The fundamental building block of any web page

CSS

   Cascading Sheet Style 		Stylize the web page
  • Applies style to web page elements
  • Targets various screen sizes to make web pages responsive
  • Primarily handles the look and feel of a web page

 JS

  Java Script

Stylize the website
  • Applies styles to the web page element
  • Targets various screen sizes to make web pages responsive
  • Primarily handles the look and feel of a web page
